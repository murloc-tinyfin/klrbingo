Kirby live radio bingo

Rules:

  1. You have to gain 300 xp on top of the xp you already have on the leaderboard for each card that contains that space. When you achieve the 300 xp you must annouce it to the chat.
  2. For matching theme if a song from a game plays while that game is in the gameplay corner it checks that space, it doesn't have to be the specific song matching the level of the gameplay.
  3. For specific song remixes are allowed, but it has to be an individual remix, not a medley.
  4. One exception to the "No medley" rule is morpho knight dream friend boss fight themes, they always count.
  5. Songs from remakes of games count for the original game, so Adventure counts for Nightmare in Dreamland and vice versa.
  6. If there are are any questions or concerns let us know. Suggestions for bingo spaces are also accepted.
  7. Most importantly, have fun!
